date command epoch
=================
**epoch to reable format**
> ``# date -d @<epoch>``

**date to epoch**
> ``# date +'%s'`

**date format**
> ``# date  --date="STRING"``
> ``# date  --date="next Friday"``
> ``# date  --date="2 days ago"``

Bash debug:
===========
> ``# set -x``
> or 
> ``# bash -x <script.sh>``

## GIT Tutorials

### Git Installation

- Centos 
  ``` #yum install git ```
- Ubuntu
   ``` #sudo apt-get install git ```

### Git Configuration 

``` #git config --global user.name "user"```
``` #git config --global user.email "user@example.com"```

### Git Branch

- Create Branch
``` #git branch <branch_name>```
- Delete Branch
``` # git branch -d <branch_name> ```
	- ```-D``` Option --force delete if content not commit
